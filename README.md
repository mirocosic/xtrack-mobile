XTrack
========

React native mobile app for tracking personal expenses.
---------------

The best one.
---------------

Built using:
 - react native
 - redux
 - react-navigation
 - lots of coffee and green

Features:

  - multiple personal accounts
  - custom categories and labels
  - fontAwesome icon set

Upcoming:
  - shared accounts
  - reoccurring transactions
